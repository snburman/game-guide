# Player Guide

Welcome to the bitscrawler player guide!

#### In this guide, you will learn how to:
- Create an account
- Create drawings
- Create maps
- Link maps together
- Explore your new world

:::info
If you already have an account, you can skip to [Create a Drawing](guide-drawing/create-drawing).
:::

## Create an account

Start by navigating to [bitscrawler.com](https://bitscrawler.com) to create an account.


<img src={login1} className={"tutorialImg"}/>

## Dashboard
Once you have created an account, you will be redirected to the dashboard. 

Press or click `Create World` or the pencil icon in the menu bar.

# Create a Drawing

Let's start by creating a tile drawing to place on your map.

From the `Draw` tab in the dashboard, you can create a new drawing.
<img src={draw1} className={"tutorialImg"}/>

:::info
<img src={newIcon} className={"icon"} />
<br/>
<i>New drawing icon</i>

You can also create a new drawing and/or change the canvas size by pressing the icon in the <b>top right corner</b>.
:::

## Drawing Tools

### Color Palette
Open the color picker by clicking on the color swatch. You can choose from a variety of colors.

<img src={draw2} className={"tutorialImg"} />

### Paint Bucket <img src={bucketIcon} className={"icon"} />
Select the paint bucket tool to fill the canvas with the selected color.

:::info
Pressing the paint bucket icon will toggle between the paint bucket and pencil tools
:::

<img src={draw3} className={"tutorialImg"} />


### Brush

Select the color palette again and choose a different color.

I chose to draw some grass to use as ground tiles.

:::warning
Remember to toggle the paint bucket tool off by clicking on the icon
:::
<img src={draw4} className={"tutorialImg"} />

### Save Drawing <img src={saveIcon} className={"icon"} />

Press the save icon, pick a name for your drawing, and press `Save`.

:::info
Use the dropdown to select the type of drawing you are saving

For now, we will save it as a `Tile`
:::

<img src={draw5} className={"tutorialImg"} />

### Eraser <img src={eraserIcon} className={"icon"} />
Select the eraser tool to erase pixels from the canvas.

### Erase Drawing <img src={trashIcon} className={"icon"} />
Press the trash icon to erase the entire drawing.

This <b>does not</b> delete the drawing but clears the canvas.

### Undo <img src={undoIcon} className={"icon"} />
Press the undo icon to undo the last action.

### Redo <img src={redoIcon} className={"icon"} />
Press the redo icon to redo the last action.

### Grid <img src={gridIcon} className={"icon"} />
Press the grid icon to toggle the grid on and off.

# Create a Map

Open the drawer menu and select the `Maps` tab.
<img src={drawMenu} className={"tutorialImg"}/>

## The Map canvas
The map canvas is where you can create maps using the images you've made. You can draw tiles and objects on the canvas to create a scene.

## Add Images
Start by selecting the images icon on the bottom left.

<img src={map1} className={"tutorialImg"}/>

Select an image to place on the canvas. 
<img src={map2} className={"tutorialImg"}/>


Press on a map area to place an image on the canvas.
Cover the canvas with tiles to create a ground for your map.
<img src={map3} className={"tutorialImg"}/>


Select the door icon to place an entrance on the map. This is where the player will start when entering the map.
<img src={map4} className={"tutorialImg"}/>

<img src={map5} className={"tutorialImg"}/>

<br/>
<br/>
:::warning
Remember to save your map!
:::

## Map Tools

### Save Map <img src={saveIcon} className={"icon"} />
Save your map to your account.

### Create New Map <img src={newIconBlue} className={"icon"} />
Create a new map by pressing the icon in the top right corner.

### Edit Map <img src={folderIcon} className={"icon"} />
Once you have saved a map, you can select a map you want to edit or delete.
<img src={map6} className={"tutorialImg"}/>

### Image Selection <img src={imagesIcon} className={"icon"} /> 
Select the image you want to place on the canvas.

### Inspect Area <img src={pointIcon} className={"icon"} />
Select the area you want to inspect. From here you can remove images or adjust their position.

Here, I've added a small tree object and changed its X and Y coordinates using the sliders.
<img src={map7} className={"tutorialImg"}/>

### Erase Map <img src={trashIcon} className={"icon"} />
Erase the map canvas. This will not delete the map from your account.

### Place Entrance <img src={doorIcon} className={"icon"} />
Place an entrance on the map. The entrance is where the player will start when entering the map.

### Create Portal <img src={portalIcon} className={"icon"} />
Create a portal on the map. Portals are used to move between maps.

:::info
    You can create a portal to a map of your own or any other player
:::
:::tip
Use the search bar to find maps by name or username
:::
<img src={map8} className={"tutorialImg"}/>

Press the eye icon to preview a map.
<img src={map10} className={"tutorialImg"}/>

Select an area to place a portal. Now, when a character enters this area, they will be transported to the map you selected.
<img src={map9} className={"tutorialImg"}/>

<br/>
<br/>
:::warning
Portals and objects cannot be placed on the same tile.
:::
